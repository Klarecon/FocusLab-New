The single selected-state spec (applied to all four)

  • Indicator: pink filled circle, white ✓, 20px — replaces the orange browser checkbox and the no-indicator cards.
  • Container: 2px solid #c4186a border + rgba(196,24,106,0.05) fill + soft glow.
  • Hover (unselected): border darkens, faint tint — so it's obvious a row is clickable.
  • Hit area: whole row/card clickable, min 44px tall (easier taps, better a11y).
  • Orange stays for one job only: waste hours values — never for "selected". Pink = you chose it, orange = how much waste.
